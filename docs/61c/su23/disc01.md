## Number Representation

* Everything is bits!
* Numbers
    - 19 -> `0b00001011`
* ASCII Characters
    - Even letters and punctuation can be represented with bits
    - `a` -> `0b1100001`
* Computer Instructions
    - `3 + 0` -> `0b00000000001100000000000010010011`
